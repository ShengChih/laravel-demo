## 基於 Laravel 5.8 實作簡易 CRUD 功能（留言板）

- 實作 Controller index, create, store, destroy
- Defalut Laravel Authentication

## 需要

- PHP7.2
- MariaDB 10.3.16
- Laravel Framwork 5.8

## Step

- 設定 .env DB 資訊
- 安裝 ＭySQL/MariaDB 並啟動

```
composer i
npm i
php artisan serve --port 8000
```
